# Disease-Prediction-System
<h3>📝 Description :</h3> 
The project aims to develop a machine learning-based system to predict diseases based on user-reported symptoms. By leveraging algorithms like Random Forest, Naive Bayes, and Support Vector Machine (SVM), the system will classify diseases with high accuracy and assist in early diagnosis, improving healthcare outcomes.

<br>
Our vision is to transform healthcare by making advanced technology an integral part of everyday diagnosis and treatment.
<br>

Our mission is to bridge the gap between technology and healthcare by developing intelligent solutions that enhance medical diagnosis and decision-making.


<h3>🌐 Web Link:</h3>

<h4> Live Site: https://disease-prediction-symptoms-project.onrender.com/ </h4>

### 📃 Features :
- Healthcare Applications: Assists doctors and patients in early diagnosis.
- Scalable Framework: Can be extended to include more diseases and symptoms.
- Educational Tool: Useful for students and researchers to understand the applications of machine learning in healthcare.

### 🛠️ Technologies Used
- **Backend**: Python Flask
- **Frontend**: HTML, CSS, JavaScript, Bootstrap
- **Database**: MongoDB Atlas
- **Authentication**: JWT, OAuth 2.0
- **Encryption**: AES-256
